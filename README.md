# wtf
Web Test Framework:

Generic Test automation framework for Web automation script development.

Currently framework has integrated TestNG, GSON, Rest Assured and Cucumber library. This framework supports BDD style script development using Cucumber.

To Install:
Clone repo
Update maven

Prerequisite:
Java 8 +
Maven 3 +

To run the script need to run testng.xml in next phase will update cucumber runner as well as currently it is under development.
